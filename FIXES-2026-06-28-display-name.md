# 2026-06-28 顯示名稱修正

這版修正「編輯：guest-xxxx」出現在畫面上的問題。

## 變更內容

- 優先使用 LIFF 取得的 LINE 暱稱作為預設顯示名稱。
- 如果沒有取得 LINE 暱稱，第一次進入日記頁會要求設定顯示名稱。
- 使用者可以點上方「編輯：名稱」重新修改顯示名稱。
- 畫面上不再顯示 `guest-xxxx` 或 LINE 內部 userId。
- 行程花費的「誰先付」、建立者、最後編輯、編輯者紀錄都改成顯示家人看得懂的名稱。
- 儲存行程時會同時保存 `payerName`，避免付款人顯示成技術 ID。

## 使用方式

部署後重新打開 LINE 日記頁：

1. 如果 LIFF 有成功取得 LINE 暱稱，會直接顯示該名稱。
2. 如果沒有取得，會跳出「設定顯示名稱」。
3. 之後若要改名，點上方的「編輯：名字」即可。
