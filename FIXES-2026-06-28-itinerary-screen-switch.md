# 2026-06-28 行程新增／編輯改為明確切換畫面

這版延續 Supabase + 行程內建花費與分帳 + 已完成勾選版本，並調整行程操作流程：

- 行程分頁預設只顯示「行程清單」。
- 按「＋ 新增行程」後，會切到獨立的「新增行程」畫面。
- 新增完成後自動回到行程清單。
- 按行程上的「修改」後，會切到獨立的「修改行程」畫面。
- 修改完成後自動回到行程清單。
- 新增／修改畫面上方都有「← 回行程」，取消時會回到行程清單。
- 行程清單保留「已完成」勾選，可直接在瀏覽清單時切換完成狀態。
- 上方統計仍會顯示已完成進度，例如 `已完成 2/5`。

手機上傳 GitHub 時，請確認新版檔案覆蓋在根目錄正確位置：

- `/public/app.js`
- `/public/styles.css`
- `/src/storage.js`
- `/src/server.js`
- `/package.json`

不要變成 `/xue-family-line-bot/public/app.js` 或 `/src/public/app.js`。
